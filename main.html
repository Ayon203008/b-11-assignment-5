Question-2:What are the difference between innerHTML,innerText and textContent

TextContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

In the DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element; this leverages the "event bubbling" mechanism where events triggered on a child element propagate up the DOM tree to its ancestors, enabling efficient event handling for dynamic content or large lists of elements.

Question-4:What is event bubbling in the DOM ?

In the DOM (Document Object Model), "event bubbling" refers to the behavior where when an event occurs on an element, it triggers not only on that element but also propagates upwards through the DOM hierarchy, firing event handlers on its parent elements until it reaches the root of the document, essentially "bubbling up" the event chain; this allows you to capture events on a parent element even if the user interacted with a child element within it.

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, you can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent . For example, to add an element, you can create it using document. createElement and then append it to a parent element using appendChild .
